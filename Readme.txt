To build and run the Angular application
1. download node js and install it
2. Create a new folder in a directory of your choosing. 
3. unzip the Application to the folder you created. and all the files will be unzipped to a new folder named Angular under the folder you created above
4. Open Command Prompt and navigate to the folder named Angular you unzipped in the previous step. 
5. From the Command Prompt, run the following command to install the project dependencies
>npm i
6. From the Command Prompt, run the following command to start applicaion
>npm run start
7. run it in browser
http://localhost:4200/?lang=en
